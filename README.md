## My Personal Portfolio & Blog

### Overview
This is the source code for my personal website, a place to showcase my portfolio and share my thoughts. The site is built with **Hugo**, a fast static site generator, and uses the minimalist and modern **PaperMod** theme.

### Features
* **Portfolio:** A dedicated section to display my projects with detailed descriptions and links.
* **Blog:** A space for articles and posts on topics I'm passionate about.
* **Responsive Design:** Optimized for a great viewing experience on all devices.
* **Fast Performance:** Built as a static site for incredible speed and security.

### Technologies
* **Hugo:** The core static site generator.
* **PaperMod:** The theme used for styling and layout.
